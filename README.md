# SwasthyaSathi

## IEEE TechForGood 2026 Internship Project

SwasthyaSathi is a preventive healthcare and health accessibility platform designed to improve healthcare awareness, promote preventive healthcare practices, and simplify access to healthcare resources.

## Problem Statement

The gap between available healthcare information and public understanding leads to poor preventive healthcare practices, delayed interventions, and underutilization of healthcare resources.

## Objectives

- Improve health awareness
- Promote preventive healthcare
- Provide personalized health recommendations
- Improve healthcare accessibility
- Simplify healthcare information for users
