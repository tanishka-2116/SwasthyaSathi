Backend
This folder contains the backend development of SwasthyaSathi.

Technology Stack:
PHP 7/8
MySQL
PHPMailer
RESTful API

Responsibilities:
User Authentication & Management
Health Profile Management
REST API Development
Database Connectivity
Email Report Generation

File Structure:
backend/
